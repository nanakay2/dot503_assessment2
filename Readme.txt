This a simple react calculator Application

This app was created using the create-react-app bootstrap

The source code for this project can be found using this link https://github.com/nanakay2/dot503_assessment2

In order to run this locally, first clone the project then from the project terminal run npm install to install all the project dependencies.
Run npm run start to run the project locally

This project setup includes the use of github actions to automate the testing and building process whenever a push is made to the remote repository
In order to accomplish the same automation locally, run npm run build.
If you check the package.json file the build script has been altered to enable this automation
